# TIS2AA
----

### Getting started

#### Requirements:
 * Numpy
 * Database has to be downloaded from https://www.dropbox.com/s/uuiif7di6vg8wex/RNA09_FRAG.tar.gz?dl=0

````bash
 $ tis2aa.py  trna.cg.pdb  trna.log trna.aa.pdb
 $ min.py trna
````
